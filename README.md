# opendota-data-fetch
repo for fetching and processing data from opendota 

# Usage
## Install required libraries
```
make setup
```

## Create required dirs
```
make create-dirs
```

## Fetch hero data
```
make fetch-hero-data
```

## Fetch player data
```
make fetch-player-data
```

## Fetch and process data
```
make fetch-all-data
```
